# Jump Type

A fast typing game built with the Godot Engine.  
Every correct word launches you higher into the sky as you climb floating platforms in a dangerous zig-zag ascent.

One wrong letter and the run is over.

[![Click to Play](https://img.shields.io/badge/CLICK%20ME%20TO%20PLAY-itch.io-red?style=for-the-badge&logo=itchdotio)](https://ambariz.itch.io/jump-type)

**LINK :** https://ambariz.itch.io/jump-type  

---

## Gameplay

- Type random words fetched live from internet
- Every correct word triggers a jump  
- Climb floating platforms placed in a zig-zag pattern  
- Reach higher platforms before falling behind  
- Wrong letters instantly end the run  

The higher you climb, the faster and more intense the game becomes.

---

## Features

- Live random words
- Typing based jump system  
- Arcade style vertical climbing  
- Simple and responsive controls
